docker常用命令

显示所有容器

  • docker ps -a

显示所有镜像

  • docker images

删除镜像

  • docker rmi images_id/repository:tag

删除容器

  • docker rm container_id/container_name

下载镜像(tensorflow官方镜像源https://hub.docker.com/r/tensorflow/tensorflow/)

  • docker pull tensorflow/tensorflow:latest-gpu-jupyter(最新tensorflow-gpu+jupyter)

根据images创建并运行容器(相当与docker create + docker start)

  • docker run -it -v /home/aorus/docker/tf23:/home/mount/ --runtime=nvidia -p 8888:8888 --name first image_id program
    • -d: 后台运行
    • -v: 目录共享
    • -i: 交互式命令
    • -t: 分配伪终端
    • --rm: 运行后删除
    • --runtime=nvidia: gpu支持
    • -p: 分配端口映射8888为jupyter
    • --name: 重命名容器名
    • image_id: 镜像id
    • rogram: 运行的程序,常用/bin/bash手动启动jupyter

docker手动启动jupyter

  • jupyter notebook/lab --allow-root --ip=0.0.0.0 --port=8888

docker复制文件

  • docker cp container_id(或container_name):/etc/nginx /usr/local/nginx(将左侧文件或目录复制到右侧,容器内文件须在目录前加container_id或container_name)

docker运行容器

  • docker start -i container_name/container_id(-i为交互式命令形式)
  • docker stop container_name/container_id
  • docker restart -i container_name/container_id

docker容器生成镜像

  • docker commit -a "jjw" -m "mytf+jupyter" container_id/container_name repository:tag
    • -a: 作者
    • -m: 说明
    • container_id: 选择的容器id或name
    • repository: tag:仓库名+tag

docker上传网络仓库

  • docker push repository:tag
