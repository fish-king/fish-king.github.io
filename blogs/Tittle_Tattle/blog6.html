ubuntu卡登陆界面重复输密码

在训练过程中cache生成过大占满磁盘(tensorfow cache约为原图大小40倍),出现此bug,记录下该bug可能出现的原因以及解决方法,图型界面切换快捷键如下:

ctrl + alt + f1 命令登陆
ctrl + alt + f7 返回图形界面

1、主目录下.Xauthority所有者变为root,其他用户登陆无法读取.Xauthority文件

  • ls -a
  • sudo chown name:name .Xauthority
  • 检验:ls .Xauthority -l

2、误删/tmp目录,系统自动重建tmp目录,其他用户无权写入

  • cd ..  #返回home目录
  • cd ..  #返回root目录
  • sudo chmod 777 /tmp  #确保tmp目录权限为777

3、home目录到达磁盘上限(作者训练模型cache过大,拉满磁盘)

  • rm -rf 目录  #清理空间即可

4、登陆图形界面配置信息错误(清理图形配置文件,重新生成)

  • sudo –s
  • rm –rf ~/.gconf
  • rm -rf ~/gconfd
  • sudo shutdown –r now
